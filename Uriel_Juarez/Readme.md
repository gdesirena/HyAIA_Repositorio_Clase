# HyAIA – Herramientas para Análisis de Datos

**Autor:** Uriel Juárez

HyAIA es un paquete diseñado para facilitar tareas comunes de **análisis de datos**, incorporando funciones para:

- Limpieza de texto  
- Categorización automática de columnas  
- Identificación de variables binarias, categóricas y cuantitativas  
- Preparación de datos para modelos estadísticos o de machine learning
